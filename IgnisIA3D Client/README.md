## Exemplo: Integração com IgnisIA3D

O arquivo `ignisia3d_client_demo.py` mostra como o sistema consegue **verificar a disponibilidade da extensão IgnisIA3D** e **gerar modelos 3D** a partir de uma imagem guia.  
Esse módulo é responsável por conectar o IgnisAI com recursos avançados de modelagem 3D.
