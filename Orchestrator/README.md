## Exemplo: Orquestrador

O arquivo `orchestrator_demo.py` mostra como o sistema funciona como um **orquestrador**:  
ele recebe mensagens do usuário (texto ou voz), decide se a intenção é **bate-papo** ou uma **ação prática** (como gerar um modelo 3D pela extensão do software), aplica memória e personalidade, e retorna a resposta em texto e áudio.
